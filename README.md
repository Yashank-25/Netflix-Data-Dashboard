1) Project Title / Headline:-
Netflix Data Analysis Dashboard – Insights on Content, Views & Trends

2) Short Description / Purpose:-
This project is an interactive data dashboard that analyzes Netflix content to uncover insights about viewership, genres, languages, release trends, and duration. The goal is to help users quickly understand content popularity, audience preferences, and distribution across countries and languages.

3) Tech Stack:-
Power BI (for dashboard creation & visualization)
Excel / CSV (for data storage & preprocessing)
DAX (Data Analysis Expressions) (for calculations and measures)

4) Data Source:-
Netflix dataset (likely sourced from platforms like Kaggle or publicly available datasets)
Contains fields such as:
Title
Genre
Language
Country
Release Year
Duration
Views (in millions)

5) Features / Highlights:-
1. Country-wise Release Analysis
Bar chart showing total releases by country
Helps identify which countries produce the most content (e.g., USA, UK)
2. Language-based View Analysis
Area/line chart displaying total views by language
Highlights dominant languages like English, Spanish, Korean
3. Genre Distribution
Pie chart for:
Views by genre
Duration by genre
Shows which genres are most popular (Crime, Thriller, Sci-Fi, etc.)
4. Title-wise Duration Trends
Line chart showing duration of popular shows
Helps compare content length across top titles
5. KPI Cards / Filters
Top numeric cards (500, 600, 650…1200) acting as filters or key metrics
Enables quick filtering or performance tracking
6. Interactive Dashboard Design
Clean Netflix-themed UI (red & dark gradient)
User-friendly and visually appealing layout
